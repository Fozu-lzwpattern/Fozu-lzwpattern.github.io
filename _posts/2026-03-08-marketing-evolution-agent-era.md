# AI Agent 时代的营销系统演变：从工具到生命体

> 初稿 v1.2 | 2026-03-08 | 喵神出品

---

## 一、核心判断：营销正在经历第三次范式跃迁

| 时代 | 主角 | 营销逻辑 | 关键指标 |
|------|------|----------|----------|
| Web 1.0 | 网站/搜索引擎 | 流量 → 曝光 → 转化 | PV、CTR |
| 移动互联网 | App/信息流 | 算法推荐 → 千人千面 | ROI、LTV |
| **Agent 原生** | **AI代理** | **意图理解 → 自主决策 → 闭环执行** | **Agent覆盖率、自动化率、正向Token流** |

前两次变革改变的是**渠道**——从PC到手机，从搜索框到推荐流。
第三次变革改变的是**决策主体**——从人到Agent。

McKinsey 预测：到2030年，Agent主导的商业交易（Agentic Commerce）全球规模将达 **$3-5万亿**，仅美国B2C零售就有$1万亿。

---

## 二、正在发生什么：2025-2026 关键事件线

### 买方侧：消费者的Agent化

- **2025.11** — OpenAI推出"Shopping Research"，Perplexity上线"Instant Buy"一键购。Google Gemini接入实时库存数据。
- **2026.01** — Bain估算：30-45%的美国消费者已使用AI辅助购物决策。
- **2026.02** — Microsoft展示Copilot购物Agent：用户说一句需求，Agent同时理解天气、风格偏好、库存、价格、评价，完成推荐+促销+下单全链路。

**核心转变**：购物的"前门"从搜索框/首页变成了**对话**。

### 卖方侧：营销系统的Agent化

| 厂商 | 产品 | 关键能力 |
|------|------|----------|
| Salesforce | Marketing Cloud Next + Agentforce | Agent自主创建campaign、管理客户旅程 |
| Adobe | Experience Platform Agent Orchestrator | 多Agent编排、跨渠道个性化 |
| SAP | Commerce Cloud + Storefront MCP Server | 机器可读商品目录、Agent-to-Agent协议 |
| 阿里 | 瓴羊Quick Audience + 万相台 | 智能用户运营Agent |
| 深演智能 | Deep Agent | 营销Agent平台 |
| 钛动科技 | Agent+专业模型 | 出海营销Agent，$1.3亿营收 |

### 基础设施层：协议革命

- **MCP** (Model Context Protocol) — Agent读取外部数据的标准接口
- **A2A** (Agent-to-Agent Protocol) — Agent间互相协作、协商
- **AP2** (Agent Payments Protocol) — Agent自主完成支付
- **ACP** (Agentic Commerce Protocol) — Agent商业交易全流程

## 三、深层逻辑：为什么是现在？

### 1. AI公司的财务生存需要

训练前沿模型耗资数十亿，$20/月订阅远不够覆盖推理成本。**商业交易是唯一有足够体量和利润率的领域来反哺AI基础设施成本**。

### 2. 边际成本趋零的经济模型

清华报告提出"正向Token流"概念：当单次LLM调用成本 < 其驱动的经济价值时，Agent变成自我增殖的价值引擎。

### 3. 数据即界面

当AI Agent成为消费者与品牌之间的中介层，**品牌的视觉门面变得次要，结构化数据变成第一生产力**。**GEO (Generative Engine Optimization)** 正在取代传统SEO。

## 四、对营销系统的具体影响

### 4.1 营销全链路Agent化

```
选品 → 内容生成 → 投放 → 触达 → 转化 → 复购 → 分析优化
 ↓        ↓        ↓      ↓      ↓      ↓       ↓
选品     创意      投放   渠道   购物    用户    策略
Agent   Agent    Agent  Agent  Agent  Agent   Agent
```

### 4.2 "双边Agent"格局

买方有购物Agent，卖方有营销Agent。**当消费者不再"看到"你的品牌，你怎么做营销？** 答案：从"影响人的注意力"转向"赢得Agent的信任"。

### 4.3 营销自动化的进化路径

| 层级 | 描述 | 典型场景 |
|------|------|----------|
| L1 Copilot | AI辅助人类执行 | AI建议邮件标题 |
| L2 自动化 | 规则触发、批量执行 | 自动发送弃购提醒 |
| L3 自主编排 | Agent自主规划+执行+优化 | 自主创建campaign |
| L4 Agent生态 | 多Agent协作、跨系统闭环 | 营销Agent调用投放Agent |
| L5 自治系统 | 无人值守、自我进化 | 全自动内容帝国 |

**当前行业水位**：头部企业正从L2向L3迈进，L4是2026-2027的核心战场。

## 五、对美团场景的思考

- **买方侧**：外卖/到店/买菜天然适合Agent化，"问小团"AI管家方向正确
- **卖方侧**：商家经营Agent、投放Agent化、会场系统Agent化
- **基础设施**：商品/门店数据需对Agent可读，需Agent-to-Agent协议

---

## 六、几个值得深思的问题

1. 品牌忠诚度的解构：当Agent替人做决策，品牌溢价如何维持？
2. Agent中介税：Agent平台会向品牌收取多少"推荐费"？
3. 数据主权的博弈：全面拥抱第三方Agent平台，还是建立自有Agent体验？
4. 监管真空：Agent自主下单，出了问题谁负责？
5. "正向Token流"能否持续？

---

## 七、核心结论

> **营销系统的终极形态不是更好的工具，而是一个有自主意识的商业有机体。**

- **短期（2026-2027）**：L2→L3，Agent Copilot成为标配
- **中期（2027-2029）**：Agent-to-Agent商业交互成为主流
- **长期（2029-2030+）**：$3-5万亿Agent Commerce成为现实

对美团而言，关键是**能否在Agent-to-Agent的协议层占据有利位置**。
