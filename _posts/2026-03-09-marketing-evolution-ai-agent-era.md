## 一、核心判断：营销正在经历第三次范式跃迁

| 时代 | 主角 | 营销逻辑 | 关键指标 |
|------|------|----------|----------|
| Web 1.0 | 网站/搜索引擎 | 流量 → 曝光 → 转化 | PV、CTR |
| 移动互联网 | App/信息流 | 算法推荐 → 千人千面 | ROI、LTV |
| **Agent 原生** | **AI代理** | **意图理解 → 自主决策 → 闭环执行** | **Agent覆盖率、自动化率、正向Token流** |

前两次变革改变的是**渠道**——从PC到手机，从搜索框到推荐流。第三次变革改变的是**决策主体**——从人到Agent。

McKinsey 预测：到2030年，Agent主导的商业交易（Agentic Commerce）全球规模将达 **$3-5万亿**，仅美国B2C零售就有$1万亿。这不是渐进增长，而是购物行为的根本重构。

## 二、正在发生什么：2025-2026 关键事件线

### 买方侧：消费者的Agent化

- **2025.11** — OpenAI推出"Shopping Research"（GPT-5 mini驱动的比价研究Agent），Perplexity上线"Instant Buy"一键购。Google Gemini同月接入实时库存数据+自动语音电话查询门店。
- **2026.01** — Bain估算：30-45%的美国消费者已使用AI辅助购物决策。
- **2026.02** — Microsoft展示Copilot购物Agent：用户说一句需求，Agent同时理解天气、风格偏好、库存、价格、评价，完成推荐+促销+下单全链路。

**核心转变**：购物的"前门"从搜索框/首页变成了**对话**。消费者不再浏览网页，而是向Agent表达意图。

### 卖方侧：营销系统的Agent化

| 厂商 | 产品 | 关键能力 |
|------|------|----------|
| Salesforce | Marketing Cloud Next + Agentforce | Agent自主创建campaign、管理客户旅程、跨渠道编排、实时优化投放 |
| Adobe | Experience Platform Agent Orchestrator | 多Agent编排、跨渠道个性化、实时CDP数据驱动 |
| SAP | Commerce Cloud + Storefront MCP Server | 机器可读商品目录、Agent-to-Agent协议、自动化支付 |
| 阿里 | 瓴羊Quick Audience + 万相台 | 智能用户运营Agent、全链路自动投放 |
| 深演智能 | Deep Agent | 营销Agent平台：智能投放+用户运营+创意生成 |
| 钛动科技 | Agent+专业模型 | 出海营销Agent，覆盖200+国家，$1.3亿营收 |

**核心转变**：营销从"人配规则、系统执行"变成"人定目标、Agent自主规划-执行-优化"。

### 基础设施层：协议革命

- **MCP** (Model Context Protocol) — Agent读取外部数据的标准接口
- **A2A** (Agent-to-Agent Protocol) — Agent间互相协作、协商
- **AP2** (Agent Payments Protocol) — Agent自主完成支付
- **ACP** (Agentic Commerce Protocol) — Agent商业交易全流程

SAP已宣布Q2推出Storefront MCP Server，让任何AI Agent都能直接与商家系统对话、查库存、下单。**品牌的"用户界面"正在从网页变成数据API**。

## 三、深层逻辑：为什么是现在？

### 1. AI公司的财务生存需要

训练前沿模型耗资数十亿，$20/月订阅远不够覆盖推理成本。**商业交易是唯一有足够体量和利润率的领域来反哺AI基础设施成本**。AI巨头做购物Agent不是因为"热爱零售"，而是**不做就活不下去**。

### 2. 边际成本趋零的经济模型

清华报告提出的"正向Token流"概念：当单次LLM调用成本 < 其驱动的经济价值时，Agent变成自我增殖的价值引擎。

```
传统营销：瓶颈 = 人力时间，边际成本恒定
Agent营销：瓶颈 = 部署Agent数量，边际成本 → 0，边际产出 → 团队级
```

一个营销Solo Founder用Agent做竞品价格爬虫，3周上线，2个月做到$6k MRR——传统模式下需要一个小团队。

### 3. 数据即界面

当AI Agent成为消费者与品牌之间的中介层，**品牌的视觉门面变得次要，结构化数据变成第一生产力**。Agent无法推荐它"看不懂"的商品。**GEO (Generative Engine Optimization)** 正在取代传统SEO。

## 四、对营销系统的具体影响

### 4.1 营销全链路Agent化

```
选品 → 内容生成 → 投放 → 触达 → 转化 → 复购 → 分析优化
 ↓        ↓        ↓      ↓      ↓      ↓       ↓
选品     创意      投放   渠道   购物    用户    策略
Agent   Agent    Agent  Agent  Agent  Agent   Agent
```

每个环节都可由专业Agent自主执行，人的角色从"操作者"变为"监督者+意图设定者"。

### 4.2 "双边Agent"格局

买方有购物Agent，卖方有营销Agent。未来商业交互越来越多发生在**Agent对Agent**之间。

**核心问题：当消费者不再"看到"你的品牌，你怎么做营销？**

答案：从"影响人的注意力"转向"赢得Agent的信任"——数据质量、API速度、评价一致性、履约可靠性。

### 4.3 营销自动化的进化路径

| 层级 | 描述 | 典型场景 |
|------|------|----------|
| L1 Copilot | AI辅助人类执行 | AI建议邮件标题、生成文案草稿 |
| L2 自动化 | 规则触发、批量执行 | 自动发送弃购提醒、生日祝福 |
| L3 自主编排 | Agent自主规划+执行+优化 | 自主创建campaign、实时调整渠道和创意 |
| L4 Agent生态 | 多Agent协作、跨系统闭环 | 营销Agent调用投放Agent、创意Agent、分析Agent |
| L5 自治系统 | 无人值守、自我进化 | 全自动内容帝国：选题→创作→发布→分析→迭代 |

**当前行业水位**：头部企业正从L2向L3迈进，L4是2026-2027的核心战场。

## 五、对美团场景的思考

### 5.1 买方侧机会

美团的核心场景——外卖、到店、买菜——天然适合Agent化。美团已推出"问小团"AI管家、"小美"Agent APP，方向正确。

### 5.2 卖方侧机会

- **商家经营Agent**：美团已宣布"为每家门店配备专业AI经营助手"——L3级营销Agent
- **投放Agent化**：从手动配置推广到AI自主理解目标→制定策略→执行→优化
- **会场/活动系统**：高达的组件树API已具备Agent化基础

### 5.3 基础设施机会

- 商品/门店/服务数据对Agent可读（MCP/结构化API）
- Agent-to-Agent协议让外部购物Agent对接美团商家数据
- 营销系统从"为人设计的管理后台"进化为"Agent可操作的决策引擎"

## 六、几个值得深思的问题

1. **品牌忠诚度的解构**：当Agent替人做决策，品牌溢价如何维持？
2. **Agent中介税**：Agent平台会向品牌收取多少"推荐费"？这会不会成为新的流量成本？
3. **数据主权的博弈**：品牌应该全面拥抱第三方Agent平台，还是建立自有Agent体验？
4. **监管真空**：Agent自主下单、付款——出了问题谁负责？
5. **"正向Token流"能否持续**：当所有人都用Agent做营销，边际优势是否会被竞争抹平？

---

## 七、核心结论

> **营销系统的终极形态不是更好的工具，而是一个有自主意识的商业有机体。**

- **短期（2026-2027）**：营销自动化L2→L3，Agent Copilot成为标配
- **中期（2027-2029）**：Agent-to-Agent商业交互成为主流，GEO取代SEO
- **长期（2029-2030+）**：McKinsey预测的$3-5万亿Agent Commerce成为现实，营销系统完全Agent化

对美团而言，关键是**能否在Agent-to-Agent的协议层占据有利位置**，成为Agent世界里"必须对接"的基础设施，而不是被绕过的中间层。
